Project Requirements
•Objectives
The project for CSD 3156 Part 2 Cloud Computing is to assess the students’ knowledge of 
cloud computing and ability to develop cloud application services by utilising various 
cloud technologies and services from public clouds , e.g., AWS, GCP, Azure, etc . 
•Requirements  
•Students are required to develop a cloud -based web application which can interact with users 
to provide information or complete certain tasks over the Internet.
•The application shall be functional (**), scalable (*), reliable (*), elastic  (*) and secure (*). The 
number of stars indicates the priority of each feature in this work.
•Students can work in a group  (≤6 students). Submit a proposal and indicate the team 
members at https://bit.ly/3vwA1oZ by the end of Week 11. 
•Students shall submit a report, a video demo and source code by the end of Week 13.
